## :sparkles: Project Info
* * *
- **NearCat:ch Adventure** : 오프라인 모임에서 함께 즐길 수 있는 아이스브레이킹 게임
- 개발 기간 : 2022.09.06 ~ 2022.10.16 / 2023.10.10 ~ 2023.12.29

## :open_book: Description
* * *
> **NearCat:ch Adventure**는 포스트 코로나 시대에 증가한 오프라인 모임에서 함께 즐길 수 있는 아이스브레이킹 게임 서비스 입니다.

1. For What **Situation**
    - 다양한 사교 및 모임에서 
    - 처음 만난 사람들과 분위기를 풀기 위해서
    - 쉽게 관심사, 연락처, SNS등을 교환하려고 할때

2. **Who** it's for
    - 처음 만난 사람과 대화에 어려움을 느끼는 사람 
    - 포스트 코로나 시대 증가한 모임에서 사람들과 쉽게 친해지고 싶은 사람 
    - 기존과는 다른 색다른 모바일 게임 서비스를 원하는 사람 
    - 많은 사람을 만나며 대상을 기억하는데 어려움을 느끼는 사람 


## :pushpin: Features
* * *
- 소셜 로그인 : 애플 / 구글 로그인
- 프로필 등록 / 수정 : 본인 닉네임, 관심사, 연락처 및 소셜 아이디 등록 / 수정
- 관심사 카드 공유 : 매칭 시작 후 범프 시 관심사 카드 공유
- 친구 목록 : 관심사 카드 공유 시 자동으로 친구 목록에 저장하여 정보 확인 가능
- 내 아이템: 우주선 해금 시 Star Cat:ch에서 다른 캐릭터 사용 가능
- **Star Cat:ch**
    - 게임 대기: Nearby Interaction 매칭으로 상대 탐색
    - 게임 내용: 운석을 피하면서 별을 수집하는 게임 (별의 개수 = 점수)
    - 피버 타임: 친구와 일정 거리 이상 가까워지면 생성되는 별 개수 증가
    - 랭킹: 본인과 친구의 점수를 합산하여 랭킹 산정
