## :sparkles: Project Info
* * *
- 게임명 : **FLAVITY**
- 장르 : 3D 플랫포머
- 플랫폼 : PC / Web
- 개발 기간 : 2023.07.24 ~ 2023.08.28

## :open_book: Description
* * *
> 플레이어는 중력을 상하좌우로 전환할 수 있으며, 이를 활용해 발판 위를 잘 뛰어다녀야 합니다. 중력 전환을 이용하여 다양한 장애물 코스를 통과하세요!


## :pushpin: Features
* * *
- 기본 조작
    - 이동: (W), (A), (S), (D)
    - 사격: (마우스 왼쪽 클릭)
    - 달리기: (Shift)
    - 점프: (Spacebar)
    - 중력 전환 : (Q) – 왼쪽 / (E) – 오른쪽 / (Spacebar 두 번 연타) – 위쪽
- 맵 구성 및 장애물
    - 중력 전환은 점프 후 착지할 때까지 한 번씩만 사용이 가능합니다.
    - 하얀색 발판은 일반적인 발판, 푸른색의 반투명한 발판은 일정 시간 뒤 사라지는 발판입니다.
    - 과녁은 총으로 쏘면 주변의 문을 작동시키고, 폭발물은 일정량의 총알을 맞으면 폭발합니다.
