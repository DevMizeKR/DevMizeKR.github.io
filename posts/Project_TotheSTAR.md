## :sparkles: Project Info
* * *
- 게임명 : **To the STAR**
- 장르 : 2D 사이드뷰 메트로배니아
- 플랫폼 : PC
- 개발 기간 : 2024.01.15 ~ 2025.02.13

## :open_book: Description
* * *
> 별을 찾아 쓰레기장을 떠나는 한 로봇의 이야기.


## :pushpin: Features
* * *
- 기본 조작
    - 이동 : A / D
    - 대쉬 : Space
    - 상호작용 : F
    - 무기 전환 : W
    - 공격 : 마우스 우클릭
- 게임 요소
    - 점프가 불가능한 플레이어
    - 다양한 적에 다양한 무기로 대응
    - 다양한 함정과 퍼즐로 구성된 맵을 탐험
